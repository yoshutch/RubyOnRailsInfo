Quick Tutorial on Rails

Rails has several preset sample projects that it can create with the $ rails new command. In this tutorial we are going to make a simple blog

1 Create the Rails Project

Open the terminal or command line and navigate to where you want the blog application to be then execute:

$ rails new blog

This will create the sample project structure in the current directory.

2 Test the application

$ cd blog
$ rails server

This will start the Rails web server on port 3000. Go visit http://localhost:3000

You should see a "Welcome Aboard" page. If so GREAT!

3 Hello World

Stop the server by hitting Ctr+C in the terminal window.

In order to get Rails to display the typical "Hello World" you need to create a controller and a view. To do this execute this command in the terminal:

$ rails generate controller welcome index

Rails will create several files for you. Open the "/app/views/welcome/index.erb" file with your favorite text editor and replace the code with:

<h1>Hello World</h1>

4 Set the Homepage

Now you need to tell Rails to go to this new view when we visit the root of the app ( http://localhost:3000 )

Open the "/config/routs.rb" file. This file tells Rails how to handle requests coming into the application by redirecting them to specific controllers and methods. Find the line beginning with #root and uncomment it. It should look something like this:

root "welcome#index"

This tells Rails to map requests to the root of the application to the welcome controller's index action. And the get "welcome/index" also tells Rails to map requests to the welcome controller's inddex action.

Run the $ rails server command in the terminal again and then revisit http://localhost:3000. It should now show "Hello World"

5 Continue the tutorial

Now that you know the bare backbone you can continue this same tutorial at the Rails Guides page.
